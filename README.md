# Игра "Кто украл апельсины?"

<!-- Цель создания проекта, для какого курса, предмета выполнялся проект -->
### Ассеты для игры в рамках выполнения финального задания по курсу _Дизайн компьютерных игр_
<!--  Описание проекта, чему научилась, что попробовала реализовать в рамках создания проекта -->
> Ассеты моделировались в рамках проекта для реализации главной локации игры, стояла задача сделать как основные элементы взаимодействия, так и украшения, для придания сцене атмосферы, необходимо было сохраниить узнаваемость объектов, при этом использовав стилистику low-poly

<!--  Какие технологии были использованы -->
### Технологии:
- __Blender__
- Unreal Engine 4 (для разработки игры)

---
<!--  Как использовать проект, какие особенности использования -->
Модели готовы к размещению на локации, они все сделаны в рамках одной сцены поэтому размеры у них соответствуют друг другу.
